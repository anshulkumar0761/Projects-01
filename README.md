🎮 Projects-01: Basic Number Guessing Game in Java
🔹 Overview
This project is a simple number guessing game built entirely by me using Java basics. It demonstrates how beginners can combine loops, conditionals, and user input to create an interactive console-based program.

The game randomly selects a number, and the player has to guess it within a limited number of attempts. After each guess, the program provides hints like "too high" or "too low" until the correct number is guessed.

🔹 Features
🎲 Random number generation using Math.random()

⌨️ User input with Scanner

🔁 Loop-based gameplay for multiple attempts

📢 Feedback messages guiding the player

🏆 Win/Loss conditions with clear output

🔹 Learning Outcomes
Understanding loops (do-while) and conditionals (if-else)

Using random number generation in Java

Handling user input effectively

Building confidence with basic Java projects
